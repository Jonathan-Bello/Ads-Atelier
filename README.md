# Ads Atelier

## Link del proyecto publicado

https://adsatelier.mx/

## Descripcion

Landing page y sitio comercial para Ads Atelier, estudio de joyeria fina de Andrea Diaz. El proyecto presenta colecciones, piezas custom made, informacion de contacto y enlaces directos a WhatsApp e Instagram.

## Tecnologias usadas

- Astro
- React
- Tailwind CSS
- TypeScript
- Swiper
- Remix Icon
- Web Components
- API de Instagram
- Netlify

## Requisitos

- Node.js 18+
- npm
- Acceso a las credenciales o configuracion necesaria para la integracion de Instagram

## Retos tecnicos

- Integrar un feed de Instagram dentro del sitio sin afectar la experiencia visual.
- Crear carruseles para mostrar piezas de joyeria y contenido editorial.
- Mantener una navegacion clara entre secciones comerciales, colecciones y contacto.
- Optimizar imagenes y assets para una landing visual con buen rendimiento.
